Week 5

Start met HTML

In deze les zijn we begonnen met het leren van HTML. We moesten ChatGPT vragen hoe een HTML-pagina eruit zou zien voor onze treinschermen. Zo kregen we een duidelijk voorbeeld van hoe we de structuur van onze webpagina kunnen opbouwen.

Daarnaast hebben we een GitHub-account aangemaakt om onze bestanden online te beheren en te versies bij te houden: GitHub link.
Ook hebben we een Netlify-account aangemaakt om onze website te kunnen publiceren en testen: Netlify link.

HTML van ChatGPT

Onderstaand de HTML die ChatGPT had doorgestuurd, als voorbeeld:

<!DOCTYPE html>
<html lang="nl">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Treinscherm</title>
  <style>
    body { font-family: Arial, sans-serif; background-color: #1b1b1b; color: #fff; display: flex; justify-content: center; align-items: center; height: 100vh; }
    table { width: 80%; border-collapse: collapse; }
    th, td { padding: 15px; text-align: left; }
    th { background-color: #222; font-size: 1.2em; }
    tr:nth-child(even) { background-color: #333; }
    tr:nth-child(odd) { background-color: #2a2a2a; }
    .delayed { color: #ff4d4d; font-weight: bold; }
    .on-time { color: #4dff4d; font-weight: bold; }
  </style>
</head>
<body>
  <table>
    <thead>
      <tr><th>Vertrek</th><th>Trein</th><th>Bestemming</th><th>Spoor</th><th>Status</th></tr>
    </thead>
    <tbody>
      <tr><td>12:05</td><td>IC 1234</td><td>Amsterdam Centraal</td><td>5</td><td class="on-time">Op tijd</td></tr>
      <tr><td>12:15</td><td>Sprinter 5678</td><td>Utrecht Centraal</td><td>3</td><td class="delayed">5 min vertraging</td></tr>
      <tr><td>12:25</td><td>IC 4321</td><td>Rotterdam Centraal</td><td>7</td><td class="on-time">Op tijd</td></tr>
      <tr><td>12:35</td><td>Sprinter 8765</td><td>Den Haag Centraal</td><td>2</td><td class="on-time">Op tijd</td></tr>
    </tbody>
  </table>
</body>
</html>
